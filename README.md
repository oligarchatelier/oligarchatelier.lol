# Oligarch Atelier

v1 of the single page website for Oligarch Atelier.

## Preview locally with

```bash
python -m http.server
```

Open `http://localhost:8000` in your browser.

## Simple html, css, js website

No frameworks, no dependencies, no build step.

## Features

- **Brutalist Design**: Dark mode only (`#111111`), system font stack.
- **Interactions**: Custom cursor, hover reveals, fast transitions.

## Project Structure

```
├── assets/         # SVGs and images
├── index.html      # Main markup
├── style.css       # All styles (CSS vars, dark mode, animation)
└── script.js       # Cursor logic and entrance animations
```
